# RASGrapher

## Overview
Tool for extracting and visualizing XRD data, developed primarily for easy analysis of multiple files at once. 

Primarily supports Rigaku's RAS files, but can also support INT files generated by VESTA's Powder Diffraction Pattern utility.

## Features
- Batch import of multiple XRD files
- 1D diffraction plots (automatically normalized)
- 2D colormaps (raw / log scaling)
- Automatic peak detection

## Languages / Packages Used
- Python
- PySide6
- Matplotlib

## Notes
Developed for use with the Rigaku SmartLab II in MP074 (Department of Physics, University of Toronto)
